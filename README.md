# ⚛️🌌 Quantum Arcade: 30 Quantum Computing Game Universe

A futuristic browser-based quantum simulation arcade with **30 interactive quantum games** inspired by real quantum computing concepts like superposition, entanglement, decoherence, quantum gates, probability, and quantum AI systems.

This project turns quantum mechanics into a **visual, playable, and interactive experience** using HTML, CSS, and JavaScript.

---

## 🚀 Overview

Quantum Arcade is a collection of 30 ultra-interactive quantum-inspired games that simulate core ideas of quantum computing through gameplay.

Each game helps users understand quantum concepts like:

- ⚛️ Superposition
- 🔗 Entanglement
- 📉 Decoherence
- 🎯 Measurement collapse
- 🎲 Probability distributions
- 🔁 Quantum gates
- 🧠 Quantum AI behavior
- 🧪 Circuit simulation
