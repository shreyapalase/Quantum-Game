⚡ Superposition Quantum Game

Qubit State: |ψ⟩ = α|0⟩ + β|1⟩

Probability |0⟩ = 50%

Probability |1⟩ = 50%

🎮 How to Play

1. Rotate qubit to change probability

2. Measure to collapse state

3. Try to hit target state |1⟩

🏆 Win Condition

Win if measurement = |1⟩

💀 Lose Condition

Lose if measurement = |0⟩
