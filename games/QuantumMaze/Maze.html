⚛ Quantum Maze

Rules

  • Reach the glowing exit portal.
  • Each move shifts quantum probability paths.
  • Some paths collapse (blocked states).
  • Wrong collapse = probability loss.
  • Reach exit before probability hits 0.

Qiskit Reference

from qiskit import QuantumCircuit, Aer, execute

qc = QuantumCircuit(2, 2)
qc.h(0)
qc.cx(0, 1)
qc.measure([0,1],[0,1])

backend = Aer.get_backend('qasm_simulator')
result = execute(qc, backend, shots=1024).result()
counts = result.get_counts()
print(counts)
