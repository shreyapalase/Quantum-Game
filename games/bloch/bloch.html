📘 Game Description

This simulation visualizes a single qubit on the Bloch Sphere. Your goal is to rotate the quantum state (θ, φ) until it matches the hidden target state.

🧠 How to Play

  • Adjust θ (latitude) and φ (longitude)
  • Try to match the hidden quantum state
  • Press “Measure Qubit” to collapse wavefunction
  • Precision determines success

🏆 Win / Loss Conditions

WIN: Your state matches target within tolerance (±0.15)

LOSS: Measurement collapses to incorrect basis state

🧪 Qiskit Reference (Quantum Circuit)

from qiskit import QuantumCircuit, Aer, execute
import numpy as np

qc = QuantumCircuit(1, 1)

# Prepare state
theta = np.pi / 3
phi = np.pi / 4

qc.u(theta, phi, 0, 0)

qc.measure(0, 0)

backend = Aer.get_backend('qasm_simulator')
result = execute(qc, backend, shots=1024).result()

print(result.get_counts())
