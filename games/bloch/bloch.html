⚛️ Quantum Bloch Sphere Game

A quantum state simulation challenge

📘 Game Description

You are controlling a single qubit represented on the Bloch Sphere. Rotate the quantum state and try to match the hidden target state. Then measure the qubit and hope for the correct outcome.

🎮 How to Play

  • Adjust θ (theta) and φ (phi) using sliders
  • Align the qubit close to the target state
  • Press Measure Qubit
  • Outcome depends on quantum probability

🏁 Win / Loss Condition

✔ Win: Measurement matches target outcome OR fidelity > 0.92
✖ Loss: Otherwise

💻 Qiskit Reference (Quantum Circuit)

from qiskit import QuantumCircuit, Aer, execute
from numpy import pi

qc = QuantumCircuit(1, 1)

# Prepare arbitrary state
qc.ry(theta, 0)
qc.rz(phi, 0)

qc.measure(0, 0)

backend = Aer.get_backend('qasm_simulator')
result = execute(qc, backend, shots=1024).result()
counts = result.get_counts()
print(counts)
