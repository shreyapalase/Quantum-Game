Learn quantum states through visualization + interaction

🎮 Game Panel

📘 Description

This simulator visualizes a single qubit on Bloch Sphere. Adjust θ and φ to control quantum state.

🧠 How to Play

  • Move θ and φ sliders
  • Observe Bloch vector movement
  • Try to match target state
  • Press Measure to collapse state

📜 Rules

State match is based on vector distance in Bloch space.

🏆 Win / Loss

WIN: Distance < 0.12 from target

LOSS: Otherwise collapse fails

Current Bloch Coordinates:

x=0 y=0 z=1

Result: Not measured

🧪 Qiskit Reference Code

from qiskit import QuantumCircuit
import numpy as np

qc = QuantumCircuit(1)

theta = np.pi/3
phi = np.pi/4

qc.u(theta, phi, 0, 0)

qc.measure_all()

print(qc)
