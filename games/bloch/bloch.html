📘 Game Description

This simulator models a single qubit on the Bloch sphere using quantum state equations.

🧠 How to Play

  • Adjust θ and φ sliders
  • Observe state vector movement
  • Press Measure to collapse state

🏆 Rules

Measurement outcome depends on state projection probability.

from qiskit import QuantumCircuit
import numpy as np

qc = QuantumCircuit(1)

theta = np.pi/3
phi = np.pi/4

qc.u(theta, phi, 0, 0)
qc.measure_all()

print(qc)
